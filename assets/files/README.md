# CV and downloadable files

- `Lei_Zhang_CV.tex` is the source LaTeX CV.
- `Lei_Zhang_CV.pdf` is the downloadable CV used by the website.

Rebuild PDF after editing `.tex`:

- `pdflatex -interaction=nonstopmode -halt-on-error -output-directory=assets/files assets/files/Lei_Zhang_CV.tex`

Keep this filename for downloads:

- `assets/files/Lei_Zhang_CV.pdf`
